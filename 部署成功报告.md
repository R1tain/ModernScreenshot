## 📥 用户下载使用

### 方式1: 安装程序（推荐）
1. 访问 https://github.com/R1tain/ModernScreenshot/releases
2. 下载 `ModernScreenshot_Setup_1.0.0.exe`
3. 双击安装（默认 D:\ModernScreenshot）
4. 运行程序，按 F1 开始截图

### 方式2: 绿色便携版（需安装 .NET 8.0）
1. 下载 `ModernScreenshot_Portable_1.0.0.zip`
2. 解压到任意目录
3. 运行 ModernScreenshot.exe

### 方式3: 完整便携版（无需安装任何东西）
1. 下载 `ModernScreenshot_Portable_Full_1.0.0.zip` (~70 MB)
2. 解压即用，包含 .NET 8.0 运行时

## 🔧 开发者使用

### 克隆和构建
```bash
# 克隆仓库
git clone https://github.com/R1tain/ModernScreenshot.git
cd ModernScreenshot

# 还原依赖
dotnet restore

# 编译
dotnet build -c Release

# 运行
dotnet run

# 发布（自包含）
dotnet publish -c Release -r win-x64 --self-contained true
```

### 手动触发构建
```bash
# 使用 GitHub CLI
gh workflow run "Build and Release"

# 查看构建状态
gh run list
gh run watch <run-id>
```
