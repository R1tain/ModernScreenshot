## 🚀 快速开始

### 编译和运行

```bash
cd ModernScreenshot
build.bat                # Windows 构建
# 或
dotnet build -c Release  # 手动编译
```

### 使用方法

1. 启动程序（显示在系统托盘）
2. 按 **F1** 开始截图
3. 拖动选择区域
4. 使用工具栏标注
5. 保存/复制/钉图
