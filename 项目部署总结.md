## 使用方式

### 用户使用
```bash
# 1. 下载 ModernScreenshot_Setup_1.0.0.exe
# 2. 双击安装（默认安装到 D:\ModernScreenshot）
# 3. 运行程序，按 F1 截图
```

### 开发者使用
```bash
# 克隆仓库
git clone https://github.com/R1tain/ModernScreenshot.git
cd ModernScreenshot

# 编译
dotnet build -c Release

# 运行
dotnet run
```
