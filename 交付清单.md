## 🚀 使用方法

### 编译（需要 Windows + .NET 6.0 SDK）

```cmd
# 进入项目目录
cd ModernScreenshot

# 方法1: 使用构建脚本（推荐）
build.bat

# 方法2: 手动编译
dotnet build -c Release

# 方法3: 发布单文件版本
dotnet publish -c Release -r win-x64 --self-contained -p:PublishSingleFile=true
```

### 运行
```cmd
# 开发运行
dotnet run

# 运行编译后的程序
.\bin\Release\net6.0-windows\ModernScreenshot.exe
```

### 使用
1. 启动程序（显示在系统托盘）
2. 按 F1 开始截图
3. 拖动选择区域
4. 使用工具栏标注
5. 保存/复制/钉图
