## PathManager API

### 属性

```csharp
// 应用程序根目录
PathManager.AppDirectory
// 返回: D:\ModernScreenshot

// 截图目录
PathManager.ScreenshotsDirectory
// 返回: D:\ModernScreenshot\Screenshots

// 配置目录
PathManager.ConfigDirectory
// 返回: D:\ModernScreenshot\Config

// 临时目录
PathManager.TempDirectory
// 返回: D:\ModernScreenshot\Temp
```

### 方法

```csharp
// 获取默认截图保存路径
string path = PathManager.GetDefaultScreenshotPath();
// 返回: D:\ModernScreenshot\Screenshots\截图_20260613_143025.png

// 获取配置文件路径
string configPath = PathManager.GetConfigPath("settings.json");
// 返回: D:\ModernScreenshot\Config\settings.json

// 清理临时文件
PathManager.ClearTempFiles();
```

## 使用场景

### 场景1: 标准用户
1. 运行 ModernScreenshot_Setup.exe
2. 默认安装到 D:\ModernScreenshot
3. 截图自动保存到 Screenshots 目录
4. 卸载时完全清理

### 场景2: 绿色版用户
1. 解压到任意目录（如 D:\ModernScreenshot）
2. 直接运行 ModernScreenshot.exe
3. 自动创建子目录
4. 删除目录即可清理

### 场景3: 多用户环境
1. 每个用户安装到独立目录
2. 例如: D:\ModernScreenshot_UserA
3. 数据完全独立互不干扰
